Acerca de este proyecto

Este proyecto es un ejemplo completo de una tienda de camisetas online, desarrollado para demostrar la integración de HTML, CSS, Thymeleaf y buenas prácticas de diseño web.

Objetivo del proyecto

El objetivo principal es ofrecer una plataforma sencilla y funcional donde los usuarios puedan:

  • Explorar el catálogo de camisetas.
  • Añadir productos al carrito y gestionarlo fácilmente.
  • Confirmar y revisar pedidos anteriores.
  • Registrarse y gestionar su perfil de usuario.

Tecnologías utilizadas

Para construir esta tienda se utilizaron varias tecnologías modernas y robustas:

  • HTML5: Estructura semántica de la página.
  • CSS3: Estilos y diseño responsivo.
  • Thymeleaf: Motor de plantillas para la integración con Java Spring Boot.
  • Java Spring Boot: Lógica del backend y gestión de datos.
  • Responsive Design: Adaptación a móviles y tablets.

Equipo de desarrollo

Este proyecto fue realizado por un equipo ficticio de desarrollo con roles definidos:

  • Frontend: Diseño de la interfaz, HTML y CSS.
  • Backend: Implementación de la lógica de negocio y conexión con la base de datos.
  • QA: Pruebas funcionales y de usabilidad.
  • Documentación: Creación de guías y manuales de usuario.
